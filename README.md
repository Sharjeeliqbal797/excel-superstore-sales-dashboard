# 📊 Global Superstore Sales Dashboard

## Overview

This project is an interactive sales dashboard built in **Microsoft Excel** using the **Global Superstore** dataset from Kaggle.

The dashboard provides insights into sales, profit, orders, quantity, discounts, shipping costs, product categories, customer segments, and regional performance through interactive slicers and charts.

## Features

- Interactive slicers
- KPI Cards
- PivotTables
- PivotCharts
- Monthly Sales Trend
- Sales by Region
- Sales by Category
- Sales by Segment
- Top Products Analysis
- Ship Mode Analysis
